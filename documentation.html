üìñ Introduction

The Ethiopian Date Picker is a professional, accessible JavaScript library for working with the Ethiopian calendar (Ge'ez calendar). It provides accurate conversion between Gregorian and Ethiopian dates, with full support for both Amharic and English languages.

With Event Callbacks

üìö API Reference

EthiopianDatePicker Class

Constructor

new EthiopianDatePicker(inputElement, options)
  • inputElement: CSS selector string or DOM element
  • options: Configuration object (see Configuration section)

getSelectedDate()

Get the currently selected date.

const selected = picker.getSelectedDate(); // Returns: { year, month, day, formatted, gregorian } // or null if no date selected

setDate(year, month, day)

Programmatically set a date.

picker.setDate(2017, 1, 1); // Ethiopian New Year

open()

Open the date picker programmatically.

picker.open();

close()

Close the date picker programmatically.

picker.close();

destroy()

Remove picker and clean up event listeners.

picker.destroy();

EthiopianCalendar Class

gregorianToEthiopian(date)

Convert Gregorian date to Ethiopian.

const calendar = new EthiopianCalendar(); const ethDate = calendar.gregorianToEthiopian(new Date()); // Returns: { year: 2017, month: 5, day: 4 }

ethiopianToGregorian(year, month, day)

Convert Ethiopian date to Gregorian.

const gregDate = calendar.ethiopianToGregorian(2017, 1, 1); // Returns: Date object

formatDate(year, month, day, locale, includeWeekDay)

Format Ethiopian date as string.

const formatted = calendar.formatDate(2017, 1, 1, 'am', true); // Returns: "·âÖ·ã≥·àú, ·àò·àµ·ä®·à®·àù 1, 2017"

isHoliday(month, day)

Check if a date is an Ethiopian holiday.

const holiday = calendar.isHoliday(1, 1); // Returns: { month: 1, day: 1, name: { am: '·ä•·äï·âÅ·å£·å£·àΩ', en: 'Enkutatash' } } // or null if not a holiday

getMonthName(month, locale)

Get month name in specified language.

const monthName = calendar.getMonthName(1, 'am'); // Returns: "·àò·àµ·ä®·à®·àù"

today()

Get today's Ethiopian date.

const today = calendar.today(); // Returns: { year: 2017, month: 5, day: 4 }

üí° Practical Examples

1. Date Range Restrictions

Limit selectable dates to next 30 days only:

const today = new Date(); const futureDate = new Date(); futureDate.setDate(futureDate.getDate() + 30); const picker = new EthiopianDatePicker('#datepicker', { minDate: today, maxDate: futureDate, locale: 'am' });

2. Form Integration

Integrate with HTML forms and store both Ethiopian and Gregorian dates:

<form id="myForm"> <label for="birthdate">Birth Date</label> <input type="text" id="birthdate" readonly required> <input type="hidden" id="birthdate-gregorian"> <button type="submit">Submit</button> </form> <script> const picker = new EthiopianDatePicker('#birthdate', { locale: 'am', onChange: (date) => { // Store Gregorian date for server document.getElementById('birthdate-gregorian').value = date.gregorian.toISOString(); } }); document.getElementById('myForm').addEventListener('submit', (e) => { e.preventDefault(); const ethiopian = picker.getSelectedDate(); const gregorian = document.getElementById('birthdate-gregorian').value; console.log('Ethiopian:', ethiopian); console.log('Gregorian:', gregorian); // Submit to server }); </script>

3. Date Range Selector

Create start and end date pickers that depend on each other:

<input type="text" id="startDate" readonly> <input type="text" id="endDate" readonly> <script> let startPicker, endPicker; startPicker = new EthiopianDatePicker('#startDate', { locale: 'en', onChange: (date) => { // Update end date minimum endPicker.options.minDate = date.gregorian; } }); endPicker = new EthiopianDatePicker('#endDate', { locale: 'en', onChange: (date) => { // Update start date maximum startPicker.options.maxDate = date.gregorian; } }); </script>

4. React Integration

import { useEffect, useRef } from 'react'; function EthiopianDateInput({ onChange, locale = 'am' }) { const inputRef = useRef(null); const pickerRef = useRef(null); useEffect(() => { pickerRef.current = new EthiopianDatePicker(inputRef.current, { locale, onChange: (date) => { if (onChange) onChange(date); } }); return () => { if (pickerRef.current) { pickerRef.current.destroy(); } }; }, [locale, onChange]); return ( <input ref={inputRef} type="text" placeholder="·âÄ·äï ·ã≠·àù·à®·å°..." readOnly /> ); }

5. Toggle Dark Mode

const picker = new EthiopianDatePicker('#datepicker', { darkMode: false }); function toggleDarkMode() { picker.options.darkMode = !picker.options.darkMode; if (picker.options.darkMode) { picker.container.classList.add('dark-mode'); } else { picker.container.classList.remove('dark-mode'); } }

6. Keyboard Navigation

The picker fully supports keyboard navigation:

  • Enter / Space - Open picker (when input focused)
  • Escape - Close picker
  • Arrow Keys - Navigate between days
  • Page Up/Down - Navigate between months
  • Home - Go to first day of month
  • End - Go to last day of month

üîß Troubleshooting

Picker doesn't appear

Problem: Clicking input doesn't show the picker.

Solutions:

  • Ensure CSS file is loaded before JavaScript
  • Check browser console for errors
  • Verify input element exists in DOM
  • Make sure input has readonly attribute
<!-- Correct order --> <link rel="stylesheet" href="ethiopian-datepicker.css"> <input type="text" id="datepicker" readonly> <script src="ethiopian-datepicker.js"></script> <script> new EthiopianDatePicker('#datepicker'); </script>

Dark mode not working

Problem: Setting darkMode: true doesn't apply dark theme.

Solution: Dark mode applies when the picker is opened. For immediate effect, add the class manually:

const picker = new EthiopianDatePicker('#datepicker', { darkMode: true }); // Force dark mode on container picker.container.classList.add('dark-mode');

Multiple pickers conflict

Problem: Multiple pickers on the same page interfere with each other.

Solution: Each picker is independent. Make sure each has a unique input element:

const picker1 = new EthiopianDatePicker('#date1', { locale: 'am' }); const picker2 = new EthiopianDatePicker('#date2', { locale: 'en' });

Date conversion seems off

The Ethiopian calendar is 7-8 years behind the Gregorian calendar. Conversion is based on standard Ethiopian calendar algorithms. If you notice specific dates that are incorrect, please report as an issue on GitHub.
